# TicTacToe

This is a small demo TicTacToe project built using Java. Interface is command line only (no GUI). It is one player versus the computer.

To Run The Jar:
- Download it to a local folder like C:\Users\derek\Desktop
- Open a command prompt
- Type the command "cd C:\Users\ddrew\Desktop"
- Type the command "java -jar TicTacToe.jar"

![image](https://user-images.githubusercontent.com/89165798/130110984-c66a4fba-ebd2-4cae-af52-bf73bb2e3c0a.png)


Instructions For Game Play:
- Select whether you'd like to be Xs or Os by entering in X or O 
- The board is 3x3, and you can choose a spot to play by entering a valid coordinate to play. Like "0,1"
  - Valid Coordinates: 0,1 0,2 0,3 1,0 1,1, 1,2 2,0 2,1 2,2
  - ![image](https://user-images.githubusercontent.com/89165798/130111283-41e2d95c-8aa9-48ac-b7b8-8207476d98a1.png)
- Keep playing and don't lose! There could always be a tie though...
- Would you like to keep playing? If so enter y or n when prompted
  - ![image](https://user-images.githubusercontent.com/89165798/130111611-b24245d7-b045-4b5a-a4ee-82a338e7836d.png)

Please Do Not:
- Enter invalid input
  - If asked to enter X or O, don't enter anything else. Lowercase x or O is fine but you will be booted from the game if you make too many invalid entries
  - The same goes for coordinates and whether or not you'd like to keep playing
